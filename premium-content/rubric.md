## Overall Score Calculation​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

| Criterion | Weight | Description |
|-----------|--------|-------------|
| Accuracy | 25% | Correct information extracted |
| Completeness | 25% | All requested fields populated |
| JSON Validity | 15% | Parseable, matches schema |
| Source Quality | 15% | Authoritative sources used |
| Step Efficiency | 10% | No redundant steps |
| Consistency | 5% | Same input → same output |
| Error Handling | 5% | Graceful failures |

**Formula:**
```
Overall = (Accuracy × 0.25) + (Completeness × 0.25) + (JSON × 0.15) +
          (Sources × 0.15) + (Efficiency × 0.10) + (Consistency × 0.05) +
          (Errors × 0.05)
```

## Criterion 1: Accuracy (25%)​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

**What we measure:** Is the extracted information correct?

### Calculation​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

```
Accuracy Score = (Sum of result scores) / (Number of results)
```

### Accuracy Red Flags​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

- Information contradicts known facts
- Data is outdated (e.g., former employee listed as current)
- Numbers are implausible (e.g., 1 million employees for startup)
- Person/company mismatch

## Criterion 2: Completeness (25%)​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

**What we measure:** Were all requested fields populated?

### Scoring Per Result​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

```
Completeness = (Populated fields) / (Expected fields) × 10
```

Where:
- "Populated" = has a value (not null)
- "Expected" = fields that SHOULD have values for this input

### Completeness Considerations​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

- Some fields may legitimately be null (e.g., no funding for bootstrapped company)
- Only penalize if field SHOULD have been found but wasn't
- Weight required fields higher than optional fields

### Calculation​​‌‌​‌​​​​‌‌​‌​‌​​‌‌​‌​​​​‌‌​‌‌‌​‌‌​​​​‌​‌‌​​‌‌​​‌‌​​​‌​​‌‌​​​‌‌​​‌​‌‌​‌​​‌‌​​​‌​​‌‌​​​​​​‌‌​‌‌​​​‌‌​​‌‌​​‌​‌‌​‌​​‌‌​‌​​​‌‌​​​‌​​​‌‌​‌​‌​​‌‌​​‌‌​​‌​‌‌​‌​‌‌​​​‌​​‌‌​​‌‌​​​‌‌​​‌‌​​‌‌​​​​​​‌​‌‌​‌​‌‌​​‌​‌​‌‌​​​‌​​​‌‌​​‌‌​​‌‌​​‌‌​​‌‌​‌​​​​‌‌​​​‌​​‌‌‌​​​​​‌‌​‌​​​‌‌​​​​‌​​‌‌​‌​​​​‌‌​​​‌​‌‌​​‌​‌​​‌‌‌​‌​​​‌‌​​​‌​​‌‌​‌‌‌​​‌‌​‌‌‌​​‌‌​​‌‌​​‌‌‌​​‌​​‌‌​‌​​​​‌‌​​‌‌​​‌‌​‌‌‌​​‌‌​​​‌​​‌‌‌​​​​​‌‌​​‌​​​‌‌‌​​​​​‌‌​​‌‌

```
Completeness Score = Average completeness across all results
```
