Every Claygent needs two outputs:

**Part 1: The Prompt** (instructions)
```
Given the company domain: {{domain}}

Research this company's [goal] by:

1. [Step 1 with specific source]
2. [Step 2 with fallback]
3. [Step 3 if needed]

Output to the defined fields:
- field_1: [description]
- field_2: [description]

If not found, use null. Never guess.
```

**Part 2: The JSON Schema** (for Clay)
```json
{
  "type": "object",
  "properties": {
    "field_1": {
      "type": "string",
      "description": "Description of this field"
    },
    "field_2": {
      "type": "string",
      "format": "email",
      "description": "Description of this field"
    },
    "confidence": {
      "type": "string",
      "enum": ["high", "medium", "low"],
      "description": "Confidence in findings"
    }
  },
  "required": ["field_1", "confidence"]
```
